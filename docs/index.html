两层正交性架构

flowchart LR
  A["19-cell geometry"] --> B["rho coupling matrix"]
  B --> C["IFR3 same-group masking"]
  B --> D["FFR user x subband allowedMask"]
  C --> E["linear-domain interference sum"]
  D --> E
  E --> F["SINR / capacity / edge metrics"]
  F --> G["trend assertions"]

几何层描述 19-cell 小区间距离耦合，同频层分别通过 IFR3 矩阵掩蔽和 FFR allowedMask 实现正交约束，最后在每个子带上线性累加干扰并计算指标。

FAQ

SWF 怎么求？
使用迭代互注水思路，按用户排序计算闭式水位，并用功率变化阈值和最大迭代次数控制收敛。
边缘怎么判定？
私有 MATLAB 复现按用户半径不小于 cellRadius/4 判定边缘用户。
为什么 19-cell？
中心小区加两圈邻区覆盖主要同频干扰来源，同时保持仿真规模可控。
为什么用合成数据展示？
公开层展示管线和趋势验证能力，不发布私有 MATLAB 源码或原始复现输出。
