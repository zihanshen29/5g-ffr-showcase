5G 分数频率复用仿真展示

一个公开、脱敏、基于合成数据的 5G FFR 仿真复现展示：围绕 SNR 扫描、边缘用户 SINR、容量趋势和论文风格图表生成，展示通信仿真与趋势验证能力。

查看 GitHub 仓库 方法说明 可复现说明

展示内容

网络仿真
使用合成参数构造 SNR、用户密度、容量和干扰趋势。
策略对比
比较 IFR3、SWF、FFR+IFR3、FFR+SWF 等策略的 demo 指标。
趋势验证
检查容量随 SNR 增长、FFR 提升边缘 SINR、密度提高带来干扰增加等定性规律。
论文风格图表
生成 Fig.3 / Fig.4 风格的合成 demo 图表，便于简历与面试讲解。

合规边界

该页面和仓库用于公开作品集展示。数据和图表由代码生成的 synthetic/demo 数据构成，不复制私有 MATLAB 源码，不包含原始实验输出，也不把 demo 结果表述为真实论文数据。
